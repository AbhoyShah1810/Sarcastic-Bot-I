Navigate to your server folder using the cd command:
cd sarcastic-ai-prank/server

Run the server file using Node.js:
node server.js


Navigate to your client folder:
cd sarcastic-ai-prank/client

Open the index.html file in your browser. The easiest way to do this is to use the open command on your Mac:
open index.html
